本地git仓库关联Github账号

发表于2020-08-18
|字数总计:135|阅读时长:1分钟|阅读量:

1. 查看本地配置文件

进入一个本地git仓库中,在.git文件中有一个config文件.查看是否有配置user信息

这样子的就是没有设置的

2. 查看你的github邮箱信息

选择setting

点击emails

就可以看到邮箱信息了,如果没有的就添加一个.

3. 本地设置账号信息

设置github用户名

git config –global user.name “github’s Name”

设置邮箱信息

git config –global user.email “github@xx.com“

邮箱必须要和第二步的邮箱对应
